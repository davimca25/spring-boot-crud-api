Spring Boot CRUD API

A robust REST API built with Spring Boot, following a layered architecture (Controller, Service, Repository) to ensure clean code and scalability.
Technologies

    Java 17+

    Spring Boot (Web, Data JPA)

    Hibernate (ORM)

    Flyway (Database Migrations)

    PostgreSQL (Database)

Key Features

    Layered Architecture: Clear separation of concerns for maintainability.

    Database Versioning: Automated schema management using Flyway, ensuring consistent database state across environments.

    CRUD Operations: Full lifecycle management.
